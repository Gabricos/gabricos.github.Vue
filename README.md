# gabricos.github.Vue
Progetto “Tic Tac Toe (Tris)” con Vue.js per Start2Impact

Ho creato un mini-sito del gioco Tic Tac Toe (Tris) utilizzando Vue.js, basato sulle specifiche richieste.

Ho inserito la griglia 3x3 interattiva, inserendo anche il rilevamento delle vittorie e dei pareggi e integrando un sistema di punteggio che salva i risultati nel localStorage.I punteggi vengono salvati e mantenuti anche chiudendo il browser
È possibile iniziare una nuova partita o azzerare il conteggio per iniziare una nuova parita.
